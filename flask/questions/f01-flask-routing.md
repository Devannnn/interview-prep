# How does routing work in Flask?

## Why recruiters ask this
- Checks backend framework fundamentals.
- Verifies that you understand URL-to-handler mapping clearly.

## Answer structure (keep it concise)
1. Explain that `@app.route(...)` maps an HTTP path to a Python function.
2. Show path parameters (e.g., `/documents/<id>`) and extraction in function args.
3. Mention methods (`GET`, `POST`, etc.) and route constraints.
4. Add one note on route organization for maintainability.
